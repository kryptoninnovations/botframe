Lightweight Discord.js bot framework, handles slash command register, permission checks, and event routing<br>
*"It just works"* Used by several of our own Discord bots<br><br>

Note: botframe currently uses a build number version system rather than the regular semantic<br>

## Install
```bash
npm install github:krypton-innovations/botframe#main
```
Requires `discord.js` ^14.26.4 in your project
